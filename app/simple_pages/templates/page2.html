Docker

What is Docker?

Docker is an open source platform using containerization that builds and deploys applications. It is used for developing applications, putting the applications in a container, and then shipping the container wherever it is needed to run.

What are Containers?

Containers are units of software that package and contain code for running both quickly and reliably on other computers with ease.

What are Images?

An Image is a template containing instructions for creating a container that can be run on the Docker environment. The image provides the specific instructions on how to properly run the container provided and the code within.

Docker Commands

docker build : Build an image from a Dockerfile
docker commit : Create a new image from a container's changes
docker container : Manage containers
docker rm : Remove container
docker ps : Number of current containers
docker run : Run a command in the new container
docker push : Push an image or repo to a registry
docker kill : Kill running containers
