AAA Testing

What is AAA testing?

According to developers.mews.com, "The AAA pattern is a pattern for structuring tests. It breaks each test down into three parts – Arrange, Act, and Assert – where each part is a step leading to the next. The arrange step sets up the test’s input values. The act step prompts the primary function being tested. And finally, the assert step verifies that the output of the function is what was expected."


Testing is a crucial part of the development process, as it allows for us to quickly fix issues before they get the opportunity to interfere with our finished product application. Testing methods such as AAA testing allow us to clearly structure the ways that we set up these tests. In this project for example, I created separate test files for each new feature that was implemented into the site. This way I could quickly make sure that any new features were implemented correctly and that their desired function was working properly. Within the file structure of my project directory, there are a set of testing files for each of these new features. When you are ready to test the files, go into the python terminal in the editor and input pytest to check the functionality of your simulated examples.
