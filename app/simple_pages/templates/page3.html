Testing and Pytest

Testing and Pytest are extremely integral in the development process. Testing our code allows for us to uncover bugs and issues that may hinder the user experience. Pytest is a tool that is able to test our code and provide crucial information on whether the different files passed the tests. Any file with the keyword "test" and the extension ".py" will be tested by the command "pytest" in a python terminal. In my code, I used pytest to ensure that the different simple pages I created would output the proper text to the screen with the correct references to my base templates. For each page that I created, I also created a test file that would simulate that page being run to detect if it worked properly or not.
